
# Uber Trip Analysis Dashboard

This project presents a detailed and interactive Uber Trip Analysis Dashboard built in Power BI, providing insights into booking trends, revenue metrics, user behavior, and trip patterns.

The report is divided into Overview, Time Analysis, and Details sections to help understand business performance from multiple perspectives.

## Project Highlights
### Tech Used

- Power BI for data modeling & visualization

- DAX for measures & calculated fields

- Data Cleaning (Power Query)

- CSV dataset (Trips, payment, locations, etc.)

## Dashboard Pages
### 1. Overview Analysis
Provides a high-level summary of all Uber trip metrics.

**Key KPIs**

- 103.7K Total Bookings

- $1.6M Total Booking Value

- $15 Average Booking Value

- 349K miles Total Trip Distance

- 3 miles Average Trip Distance

**Insights Included**

- Booking value by payment type

- Trip value by Day Trip vs Night Trip

- Total bookings trend by day

- Most frequent pickup and drop-off locations

- Farthest trip recorded

- Vehicle type analysis (UberX, UberXL, Uber Black, etc.)

### 2. Time Analysis
Focuses on time-based customer behavior and trip patterns.

**Visuals Included**

- Trip distance by hourly pickup trend

- Total trip distance by day of the week

- Heatmap of hourly activity for all 7 days

- Hourly load distribution

- Filters for date range and city

### 3. Details Page

A complete interactive data table with filters.

**Columns include:**

- Trip ID

- Pickup date & hour

- Vehicle type

- Payment method

- Passenger count

- Trip distance

- Booking value

- Pickup location

- Total bookings

This page is ideal for drilling down into specific entries.
